**APA 7 Reference Format Requirements:**

**General Rules:**
- Title: "REFERENCIAS" or "REFERENCIAS BIBLIOGRÁFICAS" (centered, bold)
- Start on new page
- Hanging indent: 0.5 inch (1.27 cm)
- Double-spaced throughout
- Alphabetical order by first author's surname
- No extra spacing between entries

**Common Reference Types:**

**Book:**
```
Apellido, A. A. (Año). Título del libro en cursiva. Editorial.
```

**Journal Article with DOI:**
```
Apellido, A. A., & Apellido, B. B. (Año). Título del artículo.
     Nombre de la Revista en Cursiva, volumen(número), páginas.
     https://doi.org/xxx
```

**Chapter in Edited Book:**
```
Apellido, A. A. (Año). Título del capítulo. En Editor (Ed.),
     Título del libro en cursiva (pp. xxx-xxx). Editorial.
```

**Website:**
```
Autor/Organización. (Año, Mes Día). Título del artículo.
     Nombre del Sitio. URL
```

**Recommendations:**
1. **Verify hanging indent:** First line flush left, subsequent lines indented 0.5 inch
2. **Check alphabetical order:** Sort by first author's surname
3. **Verify DOI/URL format:** No period after DOI or URL
4. **Check italics:** Book titles, journal names should be italicized
5. **Capitalization:** Only capitalize first word of title and subtitle (except proper nouns)
6. **Spacing:** Ensure double-spacing within and between entries

**Manual Review Needed:**
- Verify each reference follows exact APA 7 format
- Check for consistency in formatting
- Ensure all in-text citations have corresponding references
- Ensure all references are cited in text
