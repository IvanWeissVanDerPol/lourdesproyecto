### Agregar nuevo módulo

```
1. Crear archivo en carpeta apropiada
2. Seguir estructura estándar:
   ├─ Título con emoji
   ├─ Descripción breve
   ├─ Secciones numeradas
   ├─ Ejemplos prácticos
   ├─ Enlaces de navegación
   └─ Formato consistente

4. Actualizar ESTRUCTURA_VISUAL.md
```

### Agregar nuevo ejemplo

```
1. Crear en carpeta ejemplos/
2. Incluir:
   ├─ Contexto del ejemplo
   ├─ Documento completo
   ├─ Anotaciones explicativas
   └─ Referencias usadas

```
