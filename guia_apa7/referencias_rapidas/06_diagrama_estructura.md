## 📊 DISTRIBUCIÓN TÍPICA DE PÁGINAS

### Documento corto (ensayo, 10-15 páginas)

```
│ 1 │ Portada
│ 2 │ Resumen
│ 3-4 │ Introducción
│ 5-6 │ Desarrollo/Argumento
│ 7-8 │ Discusión
│ 9 │ Referencias
│10+│ Tablas/Figuras (opcional)
```

### Documento medio (artículo, 15-25 páginas)

```
│ 1 │ Portada
│ 2 │ Resumen
│ 3-5 │ Introducción
│ 6-9 │ Método
│10-14│ Resultados
│15-20│ Discusión
│21-23│ Referencias
│24-25│ Tablas y Figuras
```

### Documento largo (tesis, 50-200+ páginas)

```
│  i-iii │ Preliminares (portada, resumen, índice)
│ 1-15  │ Introducción y Marco Teórico
│16-30  │ Método
│31-50  │ Resultados
│51-75  │ Discusión
│76-85  │ Referencias
│86-100 │ Apéndices
```
